# Page sections and wrappers

Our two foundational components for page layout are [sections](?path=/docs/components-canvas-section--docs) 
and [wrappers](?path=/docs/components-composition-wrapper--docs). Almost 100% of the content on a page will 
sit inside these elements, which provide spacing, [theming](?path=/docs/foundations-utilities-theme--docs) 
and structure.
